🐛 TROUBLESHOOTING

Gradle sync fails?
  → File → Invalidate Caches → Restart

Build fails?
  → Build → Clean Project → Rebuild

APK won't install?
  → Enable "Unknown Sources" in device settings

App crashes?
  → Check Logcat for errors
  → Verify server is running
