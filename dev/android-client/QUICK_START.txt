TO BUILD THE APK:

Option 1 - Android Studio (Easiest):
------------------------------------
1. Download Android Studio: https://developer.android.com/studio
2. Install it
3. Open Android Studio
4. File → Open → Select "android-client" folder
5. Wait for Gradle sync (5-10 mins first time)
6. Build → Build Bundle(s) / APK(s) → Build APK(s)
7. Find APK: app/build/outputs/apk/debug/app-debug.apk

Option 2 - Already Have Android Studio:
---------------------------------------
1. Open Android Studio
2. File → Open → android-client
3. Build → Build APK
4. Done!

TEST THE APK:
-------------
1. Install on device: adb install app-debug.apk
2. Or copy to phone and install manually
3. Open app, login with: admin / admin123
4. Should connect to server if running

CONFIGURE SERVER:
-----------------
Edit: app/src/main/java/com/owltalk/api/ApiClient.kt

Change this line:
private const val BASE_URL = "http://YOUR_IP:5000/api/"

For emulator: "http://10.0.2.2:5000/api/"
For device: "http://192.168.1.115:5000/api/"
