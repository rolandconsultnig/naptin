# 🚀 How to Build Your APK

## ⚠️ Gradle Version Issue

Your system has **Gradle 4.4.1** which is too old. The project requires **Gradle 8.1+**.

### Solution 1: Use Android Studio (Recommended)

Android Studio has the correct Gradle version built-in.

1. **Open Android Studio**
2. **File → Open**
3. **Navigate to**: `/home/fes/Downloads/dev/android-client`
4. **Wait for Gradle sync** (Android Studio will download correct Gradle)
5. **Build → Build APK(s)**
6. **APK ready!**

---

### Solution 2: Upgrade Gradle Manually

If you want to use command line:

```bash
# Install Gradle 8.4 (required)
wget https://services.gradle.org/distributions/gradle-8.4-bin.zip
unzip gradle-8.4-bin.zip
sudo mv gradle-8.4 /opt/gradle-8.4
sudo ln -s /opt/gradle-8.4/bin/gradle /usr/local/bin/gradle

# Then build
cd android-client
gradle assembleDebug
```

---

### Solution 3: Use Gradle Wrapper from Android Studio

After opening in Android Studio once:

```bash
cd android-client
./gradlew assembleDebug
```
